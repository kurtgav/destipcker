# DestiPicker v2.0: High-End Glow PRD

## 1. Executive Summary

**Product Name:** DestiPcker
**Version:** 2.0 - High-End Glow
**Core Philosophy:** "Magic in Decision Making" - Ultra-premium, glowing AI assistant that creates a magical feel for mundane decisions.
**Key Visuals:** Dark mode, Glow effects, Smooth animations (Framer Motion), Shooting stars.

## 2. Core Features (Main Workflow)

### 1. Onboarding View
- **Visuals:** Cinematic entrance, high-end typography, subtle glow pulsing.
- **Action:** Single "Get Started" button with magnetic glow effect.

### 2. Auth View
- **Backdrop:** Shooting star animations (Canvas/CSS).
- **Methods:** Email / Google Login.
- **Style:** Glassmorphism cards over the starry background.

### 3. Profile Setup (Initial)
- **Fields:** Birthday, Location.
- **Visuals:** Step-by-step smooth transitions, shooting stars continue.

### 4. Home View (Activity Selection)
- **Interaction:** Horizontal/Grid scroll of glowing cards.
- **Options:** Cafe, Food, Shopping, Nature Trip, etc.
- **UX:** "Hover to glow" interactions.

### 5. Preferences (Budget & Radius)
- **Budget:** Exact amount input OR glowing smooth slider.
- **Radius:** Exact amount input OR glowing smooth slider.
- **Action:** "Decide for Me" - triggers a magical particle explosion/animation.

### 6. Decision Result
- **Output:** 2 Venues only (Curated/AI selected).
- **Criteria:** 4+ Stars, Budget Match.
- **UI:** Two high-end cards revealing themselves.

## 3. Future Features (Next Steps)
- **Outfit Picker:** Upload 2 photos -> AI picks one.
- **Menu Scanner:** Photo of menu -> AI suggests best item.

## 4. Profile & Subscription
- **Profile Management:** Update details, payment methods (GCash/Maya/Card).
- **Subscription:**
  - **Perks:** Unlimited uses (2 months), No Ads, 5 Venue suggestions (instead of 2).
  - **Actions:** Upgrade, Cancel, Delete Account.

## 5. Technical Requirements
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Custom "Glow" Utilities
- **Animation:** Framer Motion (page transitions, micro-interactions)
- **Icons:** Lucide React
- **Backend:** Supabase (Auth + DB)

## 6. Design System Specs
- **Background:** Deep space black/matcha gradients.
- **Primary Glow:** Cyan/Purple/Gold gradients suitable for "Magical" feel.
- **Typography:** Modern Sans (Inter/Outfit) with wide tracking for headers.
- **Components:** Glassmorphism, Neon borders, Smooth easing curves.
